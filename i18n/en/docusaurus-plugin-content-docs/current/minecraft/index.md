# Minecraft Server Guide

Configure and manage your Minecraft server with LKLCloud.

## Server Types

We support various Minecraft server types:

- Vanilla
- Spigot
- Paper
- Forge
- Fabric

## Getting Started

Access your server through the game panel and upload your world files.
