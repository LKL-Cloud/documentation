# FiveM Server Guide

Learn how to configure and manage your FiveM server.

## Getting Started

FiveM is a modification for Grand Theft Auto V that allows you to create custom multiplayer servers.

## Requirements

- A valid GTA V license
- FiveM server files
- Basic server configuration knowledge
