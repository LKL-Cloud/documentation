# Game Panel Guide

Learn how to use the LKLCloud game panel to manage your game servers.

## Features

The game panel allows you to:

- Start, stop, and restart your servers
- Upload and manage files
- View console logs
- Configure server settings
- Manage databases
- Schedule automatic tasks

## Getting Started

Access your game panel at:
- [games.lklcloud.fr](https://game.lklcloud.fr/)

Use the credentials provided in your welcome email.

For assistance, join our [Discord](https://discord.gg/UaHNnMarQA).
