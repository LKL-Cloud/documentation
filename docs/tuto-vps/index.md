Guide complet pour configurer et administrer votre VPS Linux sur LKL Cloud.

## Guides disponibles

- **[Se connecter en SSH](./se-connecter-ssh)** — Connexion à votre VPS via SSH depuis Windows, macOS et Linux
- **[Changer le port SSH](./changer-port-ssh)** — Sécuriser votre serveur en changeant le port SSH par défaut
- **[Installer Node.js & NVM](./installer-nodejs)** — Installer et gérer les versions de Node.js
- **[Installer Plesk](./installer-plesk)** — Panel de contrôle web pour gérer sites, bases de données et emails
- **[Installer Docker](./installer-docker)** — Conteneurisation avec Docker et Docker Compose
- **[Configurer un Reverse Proxy](./configurer-reverse-proxy)** — Nginx comme reverse proxy pour vos applications
- **[Monitoring serveur](./monitoring-serveur)** — Surveiller les performances de votre serveur
- **[Héberger un bot Discord](./heberger-bot-discord)** — Déployer un bot Discord (Node.js ou Python) sur votre VPS

## Prérequis

- Un VPS Linux actif sur LKL Cloud (Debian/Ubuntu recommandé)
- Vos identifiants de connexion (IP, utilisateur, mot de passe)
