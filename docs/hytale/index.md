# Hytale

Bienvenue dans la section dédiée aux serveurs **Hytale** sur LKLCloud.

## Prérequis

- Un serveur de jeux LKLCloud actif
- Accès au [panel de gestion](https://games.lklcloud.fr)

## Démarrage rapide

1. Connectez-vous à votre [panel game](https://games.lklcloud.fr)
2. Sélectionnez votre serveur Hytale
3. Démarrez le serveur via le bouton **Start**

## Configuration de base

Les fichiers de configuration principaux se trouvent dans le panel :

- **server.properties** — Configuration générale du serveur
- **Ports** — Vérifiez que les ports sont correctement ouverts (voir [guide des ports](/docs/information/ports))
